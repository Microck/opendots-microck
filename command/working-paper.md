# WORKING PAPER COMMAND

## OVERVIEW
Activates the `working-paper` skill to assist with drafting, editing, and managing mathematical research papers in LaTeX.

## USAGE
Run `/working-paper` to:
- Generate LaTeX templates for papers.
- Get assistance with bibliography management (BibTeX).
- Validate LaTeX syntax and best practices.
- Structure research arguments and proofs.

## IMPLEMENTATION
Invokes the `working-paper` skill.
